# Somarok

R&D project. Stores bookmarks inside a web version of book or any html page.

## Usage

1. click on extension. 
2. click add or plus icon.
3. click anywhere on the webpage to bookmark.
4. click on remove sign on extension popup.
5. click on bookmarks to remove the bookmark.

## Quick Start

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Open Chrome and navigate to `chrome://extensions/`, enable "Developer mode", and load the unpacked extension from the `dist` directory.

4. Build for production:

```bash
npm run build

## install extension
```

## Project Structure

- `src/popup/` - Extension popup UI
- `src/content/` - Content scripts
- `manifest.config.js` - Chrome extension manifest configuration
